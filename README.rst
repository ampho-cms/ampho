Requirements
------------

- `Flask`_ 1.x
- `Blinker`_ 1.x


Installation
------------

Under a `virtual environment`_ use following command:

.. sourcecode:: text

    pip install -U ampho


Documentation
-------------

Visit project page on the `Read the Docs`_.


Testing
-------

Install `tox`_,  chdir into the project directory and issue command:

.. sourcecode:: text

    tox
